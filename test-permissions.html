[!] Тест разрешений REonika

Эта страница позволяет протестировать систему запроса разрешений приложения

[i] Информация

При первом посещении приложения REonika автоматически запрашивает все необходимые разрешения для полноценной работы. Эта страница позволяет проверить текущий статус разрешений и протестировать их запрос.

[i] Что тестируется

Уведомления: Для получения сообщений когда приложение свернуто
Микрофон: Для записи и отправки голосовых сообщений
Камера: Для отправки фотографий и видеосообщений
Буфер обмена: Для быстрой вставки текста и изображений
Геолокация: Для определения местоположения (опционально)
Постоянное хранилище: Для офлайн-работы приложения
